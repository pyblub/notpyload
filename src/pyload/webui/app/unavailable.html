WebUI not available

You are running a pyLoad version without prebuilt webui. You can download a build from our website or deactivate the dev mode. If desired you can build it yourself by running:
  • Install nodejs for your OS. It's maybe already pre-installed.
  • Change to the pyload.webui directory
  • npm install
Everytime you want to test or apply your changes, you've made on the WebUI, run grunt build from the web directory.
